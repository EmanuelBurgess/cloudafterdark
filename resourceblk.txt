What is a resource block?
A resource block is an element of Terraform. Each resource block describes one or more infrastructure objects, such as virtual networks, compute instances, or higher-level components such as DNS records.


resource "aws_instance" "web" {
  ami           = "ami-a1b2c3d4"
  instance_type = "t2.micro"
}
Resource type is "aws instnace"
Resource locan name is "web". Resource will be called web eleswhere in the same Terraform script.
